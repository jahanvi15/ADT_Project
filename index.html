About This Project

Project Overview

This dashboard is the final deliverable for DSCI-D532 (Data Management for Data Science), Spring 2026. The project built a normalized star schema SQLite database from raw BLS Consumer Price Index data and FRED economic indicators, then visualized the results in this interactive web dashboard.


The database uses a star schema with two fact tables (fact_cpi, fact_economic_indicators) and four dimension tables (dim_date, dim_category, dim_area, dim_series).

Data Sources

CPI data from the U.S. Bureau of Labor Statistics (Kaggle, CC0 Public Domain). Economic indicators from FRED Economic Data. Recession periods from NBER business cycle dating.


Coverage: 1935–2017 for CPI · 1967–2017 for economic indicators · 3,289 CPI rows · 51 annual economic snapshots.
